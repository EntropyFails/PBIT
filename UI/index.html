NoHate Bot

Identity Hate Speech Detection is the automated task of detecting if a piece of text contains identity-hate speech. The NoHate bot uses a mix of custom model and a refined Cohere Model to check if incoming messages in discord contain identity-hate speech or not.

Features

Hate Speech Detection

The bot is proficient in detecting identity based hate speech. The custom model used has an accuracy of upto 90%. Depending on Paid-tier or Cohere credits, the server can also use the fine-tuned cohere model.

Real Time Performance, Deletion and Flagging of Messages

The model has real time Performance, and has ~5ms delay in answering discord messages. If Hate Speech is Detected, the model deletes the message, flags and warns the user.

Direct Message warning the User

Once identity-hate speech is detected and deleted in the server, the bot sends a custom flag message to the DMs of the associated person in discord.
